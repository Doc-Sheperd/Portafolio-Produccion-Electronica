Uso basico de Multisim

Derek Rodríguez Cancino
Mecatrónica


En esta página se muestra un pequeño instructivo para aprender a usar algunas funciones de multisim.

Octubre 15, 2025
Multisim Electrónica
Kamil Gallardo Toledo
Mecatrónica

Multisim es una herramienta de simulación electrónica que permite diseñar, probar y validar circuitos antes de armarlos físicamente.
En este apartado se muestra el uso de la interfaz, la colocación de componentes, las funciones principales y las configuraciones de simulación más útiles para las prácticas de laboratorio.

  1. Accede al sitio oficial de National Instruments (NI) y busca la sección de descargas de Multisim 14.1.
  2. Descarga el instalador completo o el paquete NI Circuit Design Suite que incluye Multisim y Ultiboard.
  3. Ejecuta el archivo descargado y selecciona la opción Install NI Circuit Design Suite 14.1.
  4. Acepta los términos de licencia y elige la carpeta de instalación (puedes dejar la predeterminada).
  5. Espera a que se copien los archivos; el proceso puede tardar varios minutos.
  6. Al finalizar, el asistente solicitará reiniciar el sistema. Presiona Finish y reinicia el equipo.
  7. Una vez encendido el equipo, abre NI Multisim 14.1 desde el menú de inicio para comenzar a trabajar.

  • En la Barra de menús se concentran las acciones principales del programa, como crear o guardar proyectos, editar esquemas, colocar componentes, ejecutar simulaciones y acceder a las herramientas de configuración.
  • La paleta de componentes nos permite buscar y seleccionar componentes por nombre o categoría (Basic, Analog, Power, Digital) para colocarlos fácilmente en el área de diseño del circuito.
  • Tomamos la hoja de esquemático como el área de trabajo principal donde se colocan, organizan y conectan los componentes del circuito para formar el diseño electrónico.
  • Nuestra barra de instrumentos nos ofrece acceso rápido a los instrumentos virtuales, como el osciloscopio, multímetro, generador de funciones y otras herramientas utilizadas para analizar y medir el comportamiento del circuito.
  • En el panel de propiedades, si se selecciona un componente, muestra sus parámetros y características principales, como valor, tolerancia y modelo SPICE, permitiendo ajustar su configuración según las necesidades del circuito
  • Empezamos creando un proyecto nuevo desde la barra de File y guardamos (tambien se pueden usar los atajos con teclas).
  • Si vamos a place/ component. Tenemos la opcion de buscar el componente por texto, haciendo más facil su reconocimiento y colocación la hoja de esquemático.
  • Para el correcto funcionamiento de la simulación, vamos a importar librería correspondiente según el tipo de componente; por ejemplo, Basic para resistencias y capacitores, o Mixed para modelos SPICE de circuitos integrados (ICs).
    Este paso es fundamental, ya que en ocaciones, a la hora ce importar los componentes, el símbolo que no aparece, entonces es necesario buscar su libreria específica.
  • Para editar las propiedades de los componentes, se hace doble clic sobre el mismo, lo cual nos dejará ver su tolerancia y otras especificaciones avanzadas como el modelo SPICE o la configuración de pines.
  • Para asignar valores a componentes como resistencias y capacitores, escribe valores claros y unidades de medida.
    Para fuentes, especifica su amplitud, offset y forma de onda según el diseño del circuito.
    Cabe aclarar que algunos componentes permiten seleccionar modelos SPICE.
  • La herramienta de "cables", se obtiene precionando Ctrl+W, esto es util para unir pines de componentes arrastrando desde un pin hasta otro.
  • Para crear etiquetas se usa la herramienta de net labels, que se ebtiene yendo a place / Label, esta herramienta se parece a los puentes, que te evitan tener conexiones bastante largas y confusas, ayudando a organizar el proyecto. Esto se puede llevar de la mano con la herramienta de bloques funcionales, ya que ambas te permiten mantener organizado.
  • Para empezar una simulación primero necesitamos un instrumento virtual (como un Oscilloscope, Multimeter, Function Generator, Power Supply, Logic Analyzer), o según sea el caso.
  • Conectamos las puntas del instrumento (en este caso un osciloscopio) a los nodos a medir.
    Al ser una herramienta virtual, podemos modificar sus parámetros de la misma forma en la que ya se ha explicado en puntos anteriores.
  • Para configurar la simulación hay que abrir el menú Simulate / Interactive Simulation Settings. Y elegimos el tipo de análisis que más nos convenga.
  • Por último, una vez teniendo lo anterios completado, toca correr la simulación con la tecla F5 o con el ícono de Run
  • Exportaciones:
    Con la simulación terminada, exportamos datos con Trace / Export / CSV, en caso de querer exportar tazas, es importante aclarar que asegurarnos que el documento tiene unidades y leyendas incluidas.
